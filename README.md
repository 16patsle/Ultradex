# vue-pokedex

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Parses Bulbapedia export and creates JSON files in /public
Runs automatically after `npm install`.
```
npm run bulbapedia
```
